# Kerbalism Contract Science Bridge

A compatibility plugin allowing Kerbalism 3.32 science experiments to satisfy Contract Configurator Field Research contracts.

## Supported versions

- Kerbal Space Program 1.12.x
- Kerbalism 3.32
- Contract Configurator 2.13.1

## Installation

Copy the GameData folder into your KSP installation.

## Source

The project uses:

- Harmony
- Reflection only
- No modifications to Kerbalism
- No modifications to Contract Configurator
